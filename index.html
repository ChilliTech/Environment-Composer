Tutorial - Matrixr

Welcome to Matrixr! This project is available on Github at: https://github.com/ChilliTech/Matrixr. The aim of this project is to provide an easy way for anyone to create some basic 3d designs from their browser - even if your computer is about as powerful as a potato!

How To Use:

How To Description
? Open & Close This Window
CTRL-S, Autosave Save
f Switch UI Mode (Dark or Light)
Click & Drag Orbit The View
Right Click & Drag Pan The View
Middle Click & Drag, Scroll Scroll The View
s & Mouse Movement Scale The Selected Object
r & Mouse Movement Rotate The Selected Object
p & Mouse Movement Positions The Selected Object
x, y, z Lock To Specific Axis When Transforming (Positioning, Rotating, Scaling)
Right Click Cancel An Object Transformation (Position, Rotation, Scale)
